## 변경 내용

### 변경
- `project-memory/` 및 `.claude/hooks/`를 선택 → **필수** 항목으로 격상
- "확장(Extension)" 섹션을 **7단계**로 승격하여 1~7단계 필수 흐름에 통합
- `index.html` hero 섹션과 Quick Start 섹션을 하나로 통합 (소개와 4-step 카드를 단일 블록에 배치)
- nav 링크 `Extension` → `project-memory`로 변경

### 반영 파일
- `RELEASE_MANAGEMENT_GUIDE.md` — 구성 요소 테이블, 디렉토리 트리, 설치 체크리스트, 조정 포인트, CLAUDE.md 템플릿, 워크플로 요약, 섹션 제목
- `index.html` — nav, hero lede, 구성 요소 배지, 체크리스트 코드블록, Extension 섹션 id/eyebrow
- `README.md` — 가이드 구성 테이블, 적용 항목 설명, 적용 순서

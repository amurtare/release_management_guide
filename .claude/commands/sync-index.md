# /sync-index

`RELEASE_MANAGEMENT_GUIDE.md`와 `README.md`의 현재 내용을 읽고, `index.html`의 해당 섹션을 일치하도록 수정하라.

## 작업 범위

- 섹션 제목, 본문 설명, 표(table) 내용 변경 반영
- 코드 블록 내용 (템플릿, 예시 코드) 변경 반영
- callout/note 박스 내용 변경 반영
- 섹션 추가 시 → 사이드바 TOC 항목도 함께 추가
- 섹션 삭제 시 → index.html과 사이드바 TOC에서도 제거

## 작업 제외

- CSS, JavaScript, 레이아웃 구조는 변경하지 않는다
- HTML 엔티티 이스케이프 규칙을 유지한다 (`&amp;`, `&lt;`, `&gt;`)

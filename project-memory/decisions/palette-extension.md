# 결정: 팔레트 확장은 변형 변수로, 새 hue 추가 금지

**날짜**: 2026-05-19

## 결정 내용

색이 6개 이상 필요할 때 `:root`에 새 hue를 추가하지 않는다.
기본 5색(`--p1`~`--p5`)의 변형 변수(dark / muted)를 사용한다.

| 변형 | 공식 | 용도 |
|------|------|------|
| `--pN-dark`  | L−0.18, C×0.7 | 완료·최종 단계, 강한 강조 |
| `--pN-muted` | L+0.08, C×0.5 | 비활성·보조 상태 |

6번째 이상은 dark 티어로 사이클 재시작: node-6 → `--p1-dark`, node-7 → `--p2-dark` …

## 이유

- hue가 늘어날수록 시각적 일관성이 깨짐
- OKLCH는 지각적으로 균등해 L/C 변형만으로도 충분한 구분감을 줌
- 기존 `oklch(0.30 0.04 245)` 하드코딩이 실제로 이 패턴을 암묵적으로 따르고 있었음 → 명시적 규칙화

## 적용 위치

- `html-design-guide.md` 섹션 1.1
- `index.html` `:root` (변수 10개 추가), SVG·JS 하드코딩 제거
